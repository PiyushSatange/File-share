# File Share Application

A full-stack file-sharing system built with React (frontend) and Express (backend).

## Folder Structure
- **file-share-backend**: Contains the backend code, APIs, and database configuration.
- **file-share-frontend**: Contains the React-based user interface.

## Features
- Share files with anyone in the world
- Secure file uploads using AWS S3.
- File sharing via in-app notifications.
- User authentication and permissions.

## Setup
1. Navigate to `file-share-backend` and run the backend.
2. Navigate to `file-share-frontend` and run the frontend.
3. Run both services simultaneously for a fully functional app.
